# Beamy Bot

Beamy Bot is a Minecraft Bedrock–focused Discord bot designed to support creators and communities.  
It provides useful slash commands for custom pack requests, server information, and community interaction.

## Features
- `/request` → Get the custom pack request form
- `/rules` → Display server rules
- `/prices` → Show price list for free & paid packs
- `/portfolio` → Showcase example packs/addons
- `/collab` → Info about partnerships
- `/mommy` →  talk to "mommy"

- and much more to come...
